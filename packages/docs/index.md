# React Component Library

A comprehensive React component library for both PC and mobile platforms.

## Features

- **Cross-platform**: Support for both PC and mobile devices
- **Shared utilities**: Platform-agnostic functions and utilities
- **TypeScript support**: Full TypeScript definitions
- **Modular design**: Easy to use and extend

## Quick Start

```bash
# Install PC components
npm install @react-component-library/pc

# Install mobile components
npm install @react-component-library/mobile
```
